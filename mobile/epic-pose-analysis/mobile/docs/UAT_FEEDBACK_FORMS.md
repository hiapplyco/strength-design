## Implementation Notes

### Google Forms Setup

**For Each Scenario Form (1-10)**:
1. Create new Google Form
2. Enable "Collect email addresses" for tracking
3. Set to "Limit to 1 response" per participant
4. Add form title: "UAT Scenario [X]: [Scenario Name]"
5. Add all questions from template above
6. Set response notification to email
7. Share link in participant guide

**For Final Survey**:
1. Create comprehensive Google Form
2. Enable email collection
3. Send link on Day 7 via email and Slack/Discord
4. Set deadline: 24 hours after sending

### Response Tracking

**Spreadsheet Setup**:
1. Link all forms to single Google Sheets workbook
2. Create tabs: Form1, Form2, ..., Form10, Final
3. Create summary tab with formulas for:
   - Completion rate per scenario
   - Average ratings per scenario
   - Common themes (manual tagging)
   - NPS calculation
4. Share with team for real-time monitoring
